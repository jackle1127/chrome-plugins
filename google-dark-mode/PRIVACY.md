# Privacy Policy - Google Docs Dark Mode

**Last updated: April 10, 2026**

## Data Collection

Google Docs Dark Mode does not collect, transmit, or share any personal data.

## Local Storage

The extension stores a single preference value (`gdm_enabled`: true/false) in `chrome.storage.local` to remember whether dark mode is enabled. This data:
- Never leaves your device
- Is not shared with any third party
- Contains no personal information

## Permissions

- **storage** - Used solely to save your dark mode on/off preference locally.
- **activeTab** - Used to send the toggle message to the active Google Docs or Sheets tab.
- **tabs** - Used to reload the active tab when you click the Refresh button after toggling.
- **Host access (docs.google.com)** - Required to inject the dark mode CSS and scripts into Google Docs and Google Sheets pages.
